Section 04
三大 風險 + 修法
按影響範圍排序。每個風險對應一個具體修法。

Risk 01   Windows TUF + SSD + Tailnet 三角單點

HIGH
問題

Windows TUF 掛掉 → Whisper 停；SSD 拔掉 / 壞掉 → 整條轉錄斷；Tailnet 不穩 → Mac 無法抓 SSD 檔案。三個硬體 / 網路元件任一失效，Gooaye 股癌分析即停止，晨報「股票研究」欄缺內容。

修法
  • Mac 本地安裝 Ollama qwen2.5:1.5b 作為校稿 fallback
  • 校稿完的 .txt 雙備份（SSD + iCloud Drive）
  • proofread 腳本加偵測：Tailnet ping 失敗 → 自動切 Mac Ollama
  • 預計工時：半天

Risk 02   generate_stockcast.py 4500 行單檔

MEDIUM
問題

4500 行 + 342 行未提交 dirty diff，整個站台邏輯塞在一支腳本。改 Capitol Trades 解析可能意外破壞 TWSE 渲染。沒有 smoke test，部署後才知道壞了。技術債持續累積。

修法
  • 先提交 342 行 dirty diff（清乾淨再動）
  • 拆分：fetchers/ renderers/ mailers/ 三目錄
  • 加 smoke test：每次 deploy 前跑一輪 mock 抓取
  • 排程每日 smoke test，失敗 → Telegram 告警
  • 預計工時：2 天

Risk 03   MacroMicro 不穩 + Trump /trump 無洞察

LOW-MED
問題

MacroMicro 4 天中 3 天 stale，self-heal 已啟動但仍不穩定，晨報「總經 M 平方」欄洞察斷層。Trump /trump 頁目前只是 Federal Register 清單，沒有 AI digest / 板塊評分 / 內幕關聯，資訊密度低。

修法
  • MacroMicro：找替代數據源（FRED / 財政部開放資料）或降低在晨報的權重
  • Trump Phase 2：Gemini digest → 21b
  • Trump Phase 3：內幕關聯分析 → 21c
  • Trump Phase 4：板塊評分 → 21d
  • 預計工時：3 天（分批跑）
