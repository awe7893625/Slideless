    ├─ auto_assign (TASK_ROUTING + AdaptiveRouter)   │
    └─ severity_level validation                     │

  任務執行 ──────────────────────────────────────────┤
    ├─ planning_phases tracking (P1/P2/P3)           │

                                                     │
  任務完成 ──────────────────────────────────────────┤

    │   └─ Major/Critical without review → warn log  │
    ├─ G12: _check_model_escalation()                │
    │   └─ Higher tier than expected → warn log      │
    └─ PostMortem record                             │
                                                     │
  夜班審計 ──────────────────────────────────────────┤
    ├─ governance-audit.py (11 modules)              │
    ├─ governance-autofix.py (L1/L2/L3)             │
