S01 · Investment Thesis

SCQA — Microsoft 是企業 AI 化浪潮最穩定的受益者

Azure + Copilot 雙引擎推動 FY2026 加速增長；Base PT 個人估算 $490（+16%），EV $468（+11%）

S · Situation（現況）

全球企業 IT 支出正在從傳統軟體轉向 AI 整合平台。Microsoft 控制全球最大企業軟體生態（Office / Windows / Azure / LinkedIn / GitHub），擁有超過 3 億 M365 商業用戶，是企業 AI 化的天然入口。

C · Complication（問題）

Azure 增速從 2022 年 50%+ 降至 2025 年 33%。市場疑問：(1) Azure AI 服務是真實需求還是試用期？(2) Copilot 月費 $30 的付費轉化率是否能達到 30%+？(3) Google Cloud 和 AWS 競爭是否蠶食份額？

Q · Question（關鍵問題）

Microsoft 能否在 FY2026-2027 重新加速至 Azure 35%+ 增速，同時讓 Copilot 成為 $10B+ ARR 業務？

A · Answer（我們的答案）

Base Case：Azure 受益於 OpenAI API 呼叫量爆發（GPT-4o、o3 等），AI 服務占 Azure 增量的 30%+；M365 Copilot 在大型企業逐步從試用轉正式部署，FY2027E Revenue $370-390B，P/E 28×，PT $490。
