03 — Vercel AI Proxy

安全 API 閘道：三層防護

https://website-ebon-seven-31.vercel.app/api/ai-proxy — License 驗證 + Extension Secret + Claude Haiku

Layer 1 — Extension Secret
X-Extension-Secret header 驗證，防止任意第三方呼叫 API。無 secret → 403 Forbidden。
Layer 2 — License 驗證
Supabase licenses 表查詢，確認狀態 active + 未過期 + 綁定 device_id + Pro/Premium 方案。
Layer 3 — Claude Haiku
server-side ANTHROPIC_API_KEY，不暴露給 client。productData size 限制 2000 字元，防 token 炸彈。
無 secret → 403 ✓ 錯誤 secret → 403 ✓ 正確 secret，無 license → 401 ✓ Vercel 已部署並驗證 ✓
04 — Code Review

Quill 審查：8 項全部修復

3 個 CRITICAL + 5 個 HIGH，全部在上線前修復，Integration Test 驗證通過

嚴重度 問題描述 影響 修復方式 狀態
CRITICAL _run_post 忽略 group_ids，發佈全部社團 用戶選的社團被無視，誤發到所有社團 Post.group_ids_json + ALTER TABLE migration + 過濾邏輯 已修復
CRITICAL ai-proxy.js CORS 設為 * 任意第三方可呼叫 API，token 成本失控 限制 chrome-extension:// origin + X-Extension-Secret 驗證 已修復
CRITICAL str(images) 代替 json.dumps() 圖片路徑序列化為 Python repr，無法被 JSON.parse 還原 改為 json.dumps(data.get("images", [])) 已修復
HIGH asyncio.get_event_loop() deprecated Python 3.10+ 警告，未來版本會移除 全部改為 asyncio.get_running_loop()（4 處） 已修復
HIGH _run_post 無 timeout 保護 發文卡住時 task 永遠不結束，資源洩漏 asyncio.wait_for(timeout=3600) + 逾時自動 failed 已修復
HIGH database.py 模組載入時執行 mkdir import 有副作用，測試環境污染 移至 init_db() 函式內執行 已修復
HIGH WS callback 斷線後不清除（memory leak） 每次 WebSocket 重連都增加一個 dead callback unregister_ws_callback() + server.py finally 清除 已修復
HIGH datetime.fromisoformat() 無錯誤處理 非法日期字串造成 500 Internal Server Error try/except ValueError → HTTPBadRequest (400) 已修復
