Section 01 · 產品定義

AgentDesk 是什麼

幫 Vibe Coder 管理 AI Agent 團隊的 本地指揮台。 一行 Docker 啟動，Telegram 或 Discord 直接派任務， 用 Template Marketplace 預裝好的 Agent 配置包，帶著自己的 API Key 就能跑。

🐳

Docker 一鍵部署

本地跑，無需雲端訂閱。`docker compose up` 即啟動完整環境，後端 + 前端 + Bot Gateway 一次到位。

💬

Telegram / Discord Bot

主要交互介面。在聊天室直接輸入指令派任務，Agent 回報進度，不需要開瀏覽器。

📦

Template Marketplace

預建 Agent 配置包。content-writer、data-analyst、web-operator-team，下載即用，零設定。

🔑

BYOK — 帶自己的 Key

Bring Your Own Key。OpenAI、Anthropic、Gemini 全支援，費用自控，不經我們伺服器中轉。
