06 · Mobile App Technical Breakthrough

VPN/MITM 手機端技術突破

開源實作已存在。不需要越獄，不需要開發者模式——只需要理解 Apple 的 WiFi 定位 API 並攔截它。

①

開源基礎已存在

GitHub 上的 acheong08/ios-location-spoofer 已實作完整的 VPN 攔截方案。這不是理論——有可運行的 PoC 代碼可以直接 fork 研究。

②

NEPacketTunnelProvider 攔截原理

iOS 的 Network Extension 框架允許 App 建立本地 VPN。透過 NEPacketTunnelProvider 攔截所有流向 gs-loc.apple.com/clls/wloc 的封包——這是 Apple 的 WiFi 基站定位 API。

③

改寫 Protobuf BSSID → 假 GPS

Apple 的 WiFi 定位 API 使用 Protobuf 格式。攔截後，將真實的 BSSID（WiFi 基站 ID）對應的 GPS 座標，替換為我們想要的假座標，再回傳給 iOS 系統。iOS 本身認為這是正常的定位回應。

④

無需越獄 · 無需開發者模式

整個流程在 iOS 的正常沙盒內運行。Network Extension 是 App Store 認可的 API（VPN 類 App 都在用）。只需要正確的 entitlement 簽名。

限制：僅 WiFi 定位

VPN 攔截只能影響 WiFi 輔助定位（gs-loc.apple.com）。在戶外開闊環境下，硬體 GPS 晶片直接接收衛星信號，無法被攔截。室內場景完全有效。

解決方案：Hybrid 混合架構

iMyFone 的做法：VPN 攔截（室內 WiFi 定位）+ DVT simulate-location（戶外完整 GPS）。兩種模式互補，覆蓋所有使用場景。KongGoo 也應採用此策略。

08 · Distribution Strategy

iOS App 分發策略分析

App Store 無法上架。但有多種替代路線，每種都有可行的商業模式——關鍵是選擇正確的分發管道。

App Store

免費上架
不可行。App Store Review Guidelines 明確禁止「以欺騙方式竄改設備定位」的 App。GPS 欺騙類 App 無一例外被拒絕。即使繞過審查也會被舉報下架。

TestFlight（Beta 測試）

$99/年 Developer
同樣被拒絕。TestFlight 的 build 需要通過 Apple 審查（beta review），同樣會被識別為 GPS 欺騙 App 拒絕。無法作為公開分發管道。

Sideload（Xcode 手動安裝）

$99/年 Developer
可行，但有限制。透過 Xcode 或 Apple Configurator 將 .ipa 安裝到設備。簽名有效期 1 年（個人 Developer 帳號）。需要用戶有電腦配合首次安裝，但安裝後獨立使用。適合技術用戶的早期版本。

Enterprise Certificate（企業憑證）

$299/年 Enterprise
最佳方案 — iMyFone 使用此方式。企業開發者帳號可直接在網站提供 .ipa 下載連結（OTA 安裝），用戶掃 QR Code 即可安裝，無需電腦。憑證有效期 1 年，到期自動停用（風險點：需備多份憑證作 failover）。

AltStore / SideStore（第三方商店）

免費
可行，適合免費版。AltStore 使用用戶自己的 Apple ID 簽名，每 7 天需透過 AltStore 重新簽名（需連接電腦或 AltServer）。適合提供免費版 / 試用版，引導付費用戶升級到 Enterprise 版本。
建議策略：Enterprise Cert（主要分發）+ AltStore（免費引流） $299/年

風險評估

中等
Apple 修補 gs-loc.apple.com

Apple 更新 WiFi 定位 API 加密或改變協議，導致 MITM 攔截失效。緩解：保留 DVT 路線作為 fallback，並追蹤 iOS beta 更新。

高
Enterprise Certificate 被吊銷

Apple 隨時可吊銷企業憑證（AnyTo 曾被吊銷過數次）。緩解：同時持有多份 Enterprise Cert，輪替使用，確保至少一份有效。

中等
Apple 未來移除 DVT simulate-location

若 Apple 在 iOS 更新中移除此 API（可能性低但存在），桌面版全面失效。緩解：VPN 路線作為備援，推動用戶往手機端遷移。

中等
室外 GPS 覆蓋不足

VPN 方案僅影響 WiFi 定位，戶外 GPS 可能覆蓋。緩解：明確告知用戶使用場景（室內優先），提供 DVT 混合方案。

低
法律風險

GPS 欺騙本身屬於灰色地帶。緩解：明確 ToS 說明僅供個人娛樂用途，參考競品的免責聲明格式。
