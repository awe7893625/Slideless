風險與緩解

R1 — Claude Code 設定檔格式可能變動
緩解：我們每天使用最新版，第一時間更新 → 反而成為賣點（「永遠相容最新版」）
R2 — 免費替代品（GitHub 上的開源 CLAUDE.md）
緩解：免費的只有單檔，我們賣的是完整系統（28 agents + hooks + pipeline）= 不同層級
R3 — M1 銷量預估過於樂觀
緩解：即使 M1 只有一半（NT$17k），仍然是正現金流 + 驗證市場；Kill criteria: M2 < 30 份 → 降級為免費引流工具
