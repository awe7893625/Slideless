CH.1 · THESIS
投資論點：Memory Cycle 2026-2028 — 威剛是最直接受惠者

Silicon Motion CEO 警告 memory+SSD 短缺持續到 2028，AI Server 需求帶動 HBM 及 DDR5 同步缺貨。威剛以 DRAM 模組廠身份，直接受惠漲價週期。

  • 因果鏈：AI Server 爆炸式成長 → HBM + DDR5 需求暴增 → DRAM 供給吃緊 → 報價上漲 → 威剛模組廠 ASP 提升 + 毛利率擴張
  • 驗證證據：Q1 2026 累計營收 NT$261.1 億，DRAM 佔比 68% 為核心受惠段，SSD 25% 受 AI PC 推升
  • 市場誤解：市場低估 AI Server 對 DRAM 模組需求的直接帶動；模組廠 beta 值高於上游晶片廠（漲價 10% → 毛利率改善更大）
  • 主要風險：memory cycle 反轉（Samsung/SK Hynix 殺價策略）、消費性需求疲弱
Situation
DRAM + NAND 供給持續吃緊，AI Server 需求爆發。Silicon Motion CEO 明確警告短缺至 2028。
Complication
三星 / SK Hynix 殺價風險、消費性端（PC/手機）需求復甦時程不確定、威剛品牌溢價有限。
Answer（結論）
EV NT$487 > 現價，Memory cycle 多頭前景明確。BUY，監控 DRAM 報價月度走勢。

CH.7 · RISKS + CATALYSTS
三大風險 × 四大催化劑
▼ 下行風險
R1 · Memory Cycle 提前反轉（6-12 個月）
若 DRAM 報價跌 >15% 且連續 2 個月，Bear case 機率升至 50%。三星 aggressive 定價策略是最大觸發點。Stop-loss NT$360 為 Bear 下緣。
R2 · 消費端需求疲弱（持續觀察）
PC/手機換機週期延後、庫存去化不如預期，拖累消費 DRAM 佔比。AI Server 段無法完全覆蓋消費端缺口。
R3 · Kingston / 宇瞻競爭加劇（中期）
模組廠同質化高，若競爭對手以低價策略搶佔 AI Server 認證，威剛市占承壓。
