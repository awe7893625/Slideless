GetStarted Page

快速上手 — 7 步驟

Card variant="dashboard" 承載步驟卡。Step 4 標記 active（橘色數字 + executing badge）。

1
啟動後端服務
執行 bash restart_simple.sh 啟動 FastAPI 後端（port 5002）
2
設定環境變數
複製 .env.example 填入 API Key 等必要設定
3
認識 Agent 團隊
了解 28 個 Agent 的職責分工與觸發條件
4
建立你的第一個任務 進行中
在任務看板點擊「新增任務」，設定 auto_start: true 立即執行
5
啟動 3-Phase 規劃
Major/Critical 任務需 Scout → Ray 審查
6
查看成本報告
追蹤 token 用量、API 成本與 AI 槓桿倍數
7
設定夜班自動化
配置 LaunchAgent，Noel 每晚自動執行維護
