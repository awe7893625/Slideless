Section 04
安全問題修復紀錄
🛡
Path Traversal 防護
template slug 使用 resolve() + is_relative_to() 確保路徑不逃出 templates 目錄
⚡
Rate Limit Race Condition
INCR/EXPIRE 改為原子操作：SET NX + INCR，消除 TOCTOU 競態
🔒
Task Retry 所有權驗證
加入 403 守衛：only requester can retry，防止任意用戶重試他人任務
🗝
Docker Secret Hardening
移除 hardcoded SECRET_KEY，改為 ${SECRET_KEY:?required}，部署時強制注入
🏷
Admin 路徑語義修正
/api/admin → /api/platform，消除誤導性前綴，統一命名語義
