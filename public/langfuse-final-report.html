③ Phase 2 — Real-time Span（中期）

在 llm_service.py 的每次 LLM call 前後插入 LangFuse span：

  • 即時追蹤，不等任務完成
  • 可看到 prompt 內容（輸入什麼）
  • 可追蹤 streaming 延遲
  • 需要修改所有 LLM 調用路徑（工作量中等）

④ Phase 3 — Evaluation Pipeline（長期）

等 traces 累積到 200+ 條後啟動：

  • 用 LangFuse Datasets 建立 golden set

  • Prompt A/B 測試（比較不同版本效果）
