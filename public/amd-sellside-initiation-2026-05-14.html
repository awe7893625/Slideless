S02 · Investment Thesis — 因果鏈
為什麼現在是 AMD AI 第二輪的拐點

SCQA 框架：Situation → Complication → Question → Answer

SITUATION
AMD 在 2024-2026 完成關鍵轉型：資料中心從 $1.3B/季 飆升至 $5.8B/季（YoY +57%），Server CPU EPYC 市佔達 30% record，Instinct AI GPU 從 0 開始 ramp。AMD 已進入 AI 半導體核心圈第三席（NVIDIA → Broadcom → AMD）。
COMPLICATION
NVIDIA CUDA 護城河深，Blackwell 已出貨，軟體生態黏著度極高。AMD ROCm 雖開源，但開發者 onboarding 慢、效能仍落後 CUDA 10-15%。同時 hyperscaler 自研 ASIC（Google TPU、AWS Trainium）持續分食。
QUESTION
AMD 能否在 NVIDIA 主導的 AI GPU 市場取得結構性份額？MI350 → MI400 的 roadmap 執行力能否讓 hyperscaler 從「NVIDIA-only」走向「NVIDIA + AMD dual-vendor」？
ANSWER — THESIS：BUY
是。 ① hyperscaler 主動扶持替代，NVIDIA 供應不足推動 dual-vendor；② MI350 288GB HBM3E 技術領先 B200；③ EPYC 雙引擎確保底線；④ 開源 ROCm + vLLM-ATOM 在開源 LLM 切入。Base 個人估算區間 $500，+15% upside。
因果鏈
1
NVIDIA 統治 AI GPU 但供應不足 + 客戶集中度過高 → hyperscaler 主動扶持替代方案 → 2026 起 NVDA 80% share 開始裂解，AMD 是最直接受益者
2
MI350（288GB HBM3E，比 B200 多 50%）+ Helios rack-scale system 進軍訓練市場，從 inference-only 切到全 stack 競爭
3
EPYC Server CPU 已拿下 30% 市佔（record），Turin / Venice 持續壓 Intel → 資料中心雙引擎（GPU + CPU），提供穩定底線
4
資料中心 +57% YoY 確認 AI inflection point 已過 → AMD 軌跡進入 AI 半導體核心圈，估值重評已開始但尚未完成
